# Introduction

A practical and continuously evolving reference for **cyberintelligence**, focused on **OSINT, SOCMINT, Deep Web, threat intelligence and adversary behavior**.

This cheatsheet is designed to be **quick to consult**, **easy to navigate**, and **useful in real-world analysis**, incident response, and research scenarios.

## Purpose

The goal of this project is to provide:

- **Centralized knowledge base** for cyberintelligence concepts
- **Quick-reference cheatsheet** for daily analyst work
- **Structured way to organize intelligence**, from collection to analysis
- **Living document** that can evolve with new threats and techniques
- **Open source content** available for everyone at any moment
